Projects

AI Voice Assistant

The AI Voice Assistant is an intelligent virtual assistant that understands and responds to voice commands using Speech Recognition and Natural Language Processing (NLP). It can perform tasks like setting reminders, fetching weather updates, playing music, and answering questions. Integrated with APIs and machine learning, it delivers accurate and personalized responses. The project showcases the seamless integration of speech-to-text and text-to-speech technologies to create a user-friendly and efficient assistant.

Plant Leaf Disease Detection

The Plant Leaf Disease Detection system leverages deep learning to identify and classify diseases in plant leaves from images. Using Convolutional Neural Networks (CNNs), the model analyzes leaf patterns to detect abnormalities and predict specific diseases with high accuracy. This project aids farmers in early disease detection, enabling timely intervention to improve crop health and yield. It demonstrates the application of AI in agriculture for automated and efficient disease diagnosis.

Song Recommendation System

The Song Recommendation System uses machine learning to suggest personalized songs based on user preferences and listening history. By analyzing factors like genre, mood, and user behavior, the system employs algorithms such as collaborative filtering and content-based filtering for accurate recommendations. This project enhances user experience by delivering tailored music choices and demonstrates the effective use of AI in entertainment.

Image Manipulation App

The Image Manipulation App is a powerful tool that allows users to apply various image processing operations such as blurring, sharpening, sepia, and more. Built with efficient algorithms, the app enables real-time image transformation and enhancement. It supports operations like flipping, brightening, filtering, and color adjustments to provide a seamless editing experience. This project showcases the practical application of image processing techniques for creative and technical image manipulation.

Coffee Shop Website

The Coffee Shop Website is a responsive and visually appealing platform designed to showcase the coffee shop's offerings, ambiance, and services. It features an interactive menu, location details, online ordering, and customer reviews to enhance user engagement. Built with modern web technologies like HTML, CSS, and JavaScript, the website ensures a seamless experience across devices. This project highlights the effective use of web design and development to promote a local business and attract customers online.

Kanbas

Kanbas is a web-based learning management system (LMS) designed to streamline course management and enhance online learning experiences. It enables users to manage courses, assignments, and modules efficiently while providing role-based access for students and faculty. Built using modern web technologies like React and Redux, Kanbas supports dynamic rendering of course content and real-time updates. This project demonstrates the integration of robust frontend frameworks for building user-friendly and interactive educational platforms.
